## FE-dudu
- 특정 기술이나 라이브러리에 얽매이지 않고 더 올바른 방법을 고민합니다.
- 사용자 경험과 완성도 높은 시각적 요소가 서비스 성공의 기반이라고 생각합니다.
- 디자인 시스템, 서비스 사용에 도움이 되는 CSS 애니메이션을 적용하는 것에 관심이 많습니다.
