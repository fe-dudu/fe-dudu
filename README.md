## About
Seed 단계 스타트업에서 프론트엔드 엔지니어로 근무하며 MVP 개발과 핵심 서비스 구축을 주도했고 그 과정에서 15억 원 규모의 투자 유치와 정부 과제 선정에 기여했습니다. 이후 글로벌 블록체인 기업으로 자리를 옮겨 프론트엔드 엔지니어로서 제품 개발을 이어가고 있습니다.

빠른 개발과 배포를 통해 실질적인 비즈니스 가치를 만들어내는 것을 목표로 하며 비판적인 피드백을 성장의 기회로 삼습니다. 또한 따뜻한 소통을 바탕으로 한 관계 지향적인 조직 문화를 선호합니다.

문제를 해결하고 새로운 것을 탐구하는 과정 자체에서 즐거움과 동기를 얻습니다.
